# App 图标说明

## 需要的图标尺寸

为了在所有平台上正常显示，请准备以下尺寸的图标文件（PNG格式）：

### iOS 主要图标
- 1024x1024 (App Store)
- 60x60 (@2x = 120x120)
- 60x60 (@3x = 180x180)
- 40x40 (@2x = 80x80)
- 40x40 (@3x = 120x120)

### macOS 图标
- 16x16 (@1x, @2x)
- 32x32 (@1x, @2x)
- 128x128 (@1x, @2x)
- 256x256 (@1x, @2x)
- 512x512 (@1x, @2x)

## 添加图标的方法

1. 在 Xcode 中打开项目
2. 导航到 `Assets.xcassets/AppIcon.appiconset`
3. 将准备好的 PNG 图标拖拽到对应的插槽中
4. 确保勾选 "iOS" 和 "macOS" 平台

## 图标设计建议

- 简洁明了的设计
- 使用金色元素，体现"黄金生活专家"的主题
- 可以使用时钟、日历或生活相关元素
- 确保在各种背景下都清晰可见
